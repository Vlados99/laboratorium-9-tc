Autor: Veshnevskyi Vladyslav

Najpierw trzeba pobrać projekt, dalej wejść do konsoli z poziomu foldera z projektem i wpisać w konsoli
  - ```docker compose up -d```

Po uruchomieniu można sprawdzić uruchomione kontenery poleceniem
 - ```docker compose ps```
